# 📊 NHANES Body Measurements Analysis

## 📌 Project Overview

This project analyzes the NHANES Adult Body Measurements dataset using Python.

The project demonstrates Exploratory Data Analysis (EDA), BMI calculation, body ratio analysis, statistical analysis, and data visualization.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis
- BMI Calculation
- Waist-to-Height Ratio (WHtR)
- Waist-to-Hip Ratio (WHR)
- Z-score Standardization
- Pearson Correlation
- Spearman Correlation
- Histograms
- Box Plots

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab
